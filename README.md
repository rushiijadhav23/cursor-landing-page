## Colors

Defined in `style.css` via CSS custom properties (`:root`):

| Variable | Hex | Usage |
|----------|-----|--------|
| `--primary-font-color` | `#EDECEC` | Main text |
| `--background-color` | `#13120A` | Page background |
| `--secondary-background-color` | `#1B1912` | Cards, nav/footer areas |
| `--secondary-font-color` | `#988E94` | Muted text |
| `--primary-cta-color` | `#F54E00` | Links and primary buttons |
| `--hover-cta-color` | `#b74510` | CTA hover state |
| `--recent-highlight-background-color` | `#1B1914` | Highlights section background |
| `--recent-highlight-secondary-background-color` | `#201E19` | Highlight cards |
| `--toggle-mode-background-color` | `#26241F` | Toggle backgrounds |
| `--toggle-selected-background-color` | `#3A3834` | Selected toggle state |
